Curso de Python 3 - Estudonauta

Mundo 01: Fundamentos

Nesse módulo será apresentado ao aluno a linguagem de programação Python, explicadas as suas funcionalidades e os conhecimentos básicos da linguagem, sendo realizados exercícios para a fixação do conteúdo, dentre esses: a utilização de variáveis para realizar cálculos, o estudo de operadoes aritméticos, a utilização de frameworks da linguagem e muito mais!

Mundo 02: Estruturas de Controle

Neste segundo módulo, o aluno será introduzido ao conceito de uma estrutura de controle em um programa,com o principal foco em manipular o resultado de saída de um programa de acordo com o parâmetro. Conteúdos estudados: Condições, Estrutura de repetição for e while.
